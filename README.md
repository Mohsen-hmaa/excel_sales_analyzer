# Excel sales analyzer
## Description
 A Python application that reads multiple Excel sales files, calculates the total sales for each file, generates a summary Excel report, and logs any processing errors into a separate error log.
## Features 
- Read  multiple Excel files
- Calculate total sales for each file
- Generate a summary Excel report
- Manage and log processing errors
## Requirements
- Python 3.x
- openpyxl
## How to Run
1. Create a virtual environment:
```bash
python -m venv venv
```
2. Activate the virtual environment:
```bash
.\venv\Scripts\Activate.ps1
```
3. Install required packages:
```bash
pip install -r requirements.txt
```
4. Run the application:
```bash
python main.py
```
